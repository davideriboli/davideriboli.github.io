GIT - Comandi base

13 Nov 2020

Crea un nuovo repo locale:

git init

Scarica un repo online, senza submoduli:

git clone /path/to/repository

Aggiunge al proprio un repo in forma di submodulo.

git submodule add /path/to/repository/to/add/to/your

Scarica un repo online, coi submoduli:

git clone --recursive /path/to/repository

Aggiunge uno o tutti i file allo staging (index):

git add <filename>

git add *

Aggiunge tutte le modifiche al commit:

git commit -a

Come il precedente, ma senza la spina nel culo del commit message:

git commit -a --allow-empty-message

Aggiorna il master (aggiunger -f per forzare l’operazione, solo se davvero necessario):

git push origin master

Collega repo remoti a locali:

git remote add origin <server>

Elenca le modifiche in attesa di add o commit:

git status

Pulizie di primavera:

git gc --auto
