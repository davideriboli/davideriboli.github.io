# davideriboli.github.io

Questo piccolo repo contiene tutti i file del sito pubblicato attraverso le [GitHub Pages](https://pages.github.com/) e [Hugo](https://pages.github.com/) all'URL https://davideriboli.github.io.

Sito e repo mi servono soprattutto per spiegare agli studenti il funzionamento delle GitHub Pages, insieme a generatori tipo Hugo o [Jekyll](https://jekyllrb.com/) e a tenere qualche nota su ricette, trucchi e questioni che tendo a dimenticare troppo spesso. Probabilmente sono cose che interessano solo me e i miei studenti, ma l'accesso è comunque pubblico.
