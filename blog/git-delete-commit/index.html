GIT - Eliminare tutti i commit dal log di un repository

13 Nov 2020

Il modo giusto di procedere quando vuoi nuclearizzare un repo, ma non c’è verso di azzerare quella dannata, chilometrica lista di commit.

Logicamente, i comandi vanno inseriti uno per volta.

# Per prima cosa, facciamo checkout

git checkout --orphan latest_branch

# Poi aggiungiamo tutti file del nuovo repo

git add -A

# E prepariamo il primo commit del nuovo corso

git commit -am "Messaggio"

# Eliminiamo il branch

git branch -D master

# Portiamo il branch attuale ad essere il nuovo master

git branch -m master

# E forziamo un update

git push -f origin master

E vaffanculo anche ai chilometri di commit…
